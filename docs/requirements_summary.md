# 需求摘要

## 项目概述

数据库查询工具 - 支持用户添加数据库连接、查看元数据、执行 SQL 查询和使用自然语言生成 SQL。

## 核心功能需求

### 1. 数据库连接管理 (P1)

- 支持 MySQL 和 PostgreSQL 数据库连接
- 通过连接字符串 URL 添加数据库
- 自动获取数据库元数据（表、视图、字段）
- 支持多个数据库连接管理

### 2. SQL 查询执行 (P1)

- 提供语法高亮的 SQL 编辑器
- 仅允许 SELECT 语句执行
- 自动添加 LIMIT 1000（如未指定）
- 结果以表格形式展示
- 支持导出 CSV 和 JSON 格式

### 3. 自然语言生成 SQL (P2)

- 支持用自然语言描述查询需求
- 使用 LLM 生成对应的 SQL 语句
- 支持多种 LLM 模型切换（通义千问、Kimi）
- 生成的 SQL 可编辑后执行

### 4. 元数据管理 (P2)

- 查看表结构和字段详情
- 支持为字段添加中文备注
- 中文备注用于提高自然语言查询准确性

## 非功能需求

- 无需用户认证
- 后端 JSON 使用 camelCase 格式
- 文档使用中文
- 查询响应时间 < 5秒
